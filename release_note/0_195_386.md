# v0.195.386: デフォルトテーマを "things" に変更

## 課題

デフォルトテーマが "github" に設定されており、VSCode拡張・Electron両モードで初回起動時に github テーマが適用されていた。ユーザーの要望により、デフォルトを "things" テーマに変更する必要があった。

## 対策

### 変更箇所

1. **package.json** — VSCode設定の `any-markdown.theme` のデフォルト値を `"github"` → `"things"` に変更
2. **electron/src/settings-manager.ts** — Electronの `DEFAULTS.theme` を `'github'` → `'things'` に変更
3. **electron/src/settings-manager.ts** — Electron設定画面のテーマ選択肢に "Things" を追加（既存の選択肢に含まれていなかった）
