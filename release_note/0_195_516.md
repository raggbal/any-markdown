## 2. 左パネル開閉ボタンのバグ修正

### 課題
折り畳みボタン（≡）をクリックしてもパネルが閉じない。

### 原因
**CSSの詳細度（specificity）問題**。`init()`の初期パネル幅復元やリサイズ操作で`panelEl.style.width = 'Xpx'`がインラインstyleにセットされる。折り畳み時に`.collapsed`クラスを追加しても、CSS `.notes-file-panel.collapsed { width: 0 }` はインラインstyleより詳細度が低いため無視される。

### 対策（設計）
- 折り畳み時にインラインstyleをクリア（`panelEl.style.width = ''`）
- 展開時に`lastSavedPanelWidth`変数から幅を復元

### 対策（実装）
- **`notes-file-panel.js`**: `lastSavedPanelWidth`モジュール変数追加。`init()`で`initialPanelWidth`を保存。collapseBtn handler: `style.width = ''` + `classList.add('collapsed')`。toggleBtn handler: `classList.remove('collapsed')` + `lastSavedPanelWidth`で復元。`onPanelResizeEnd()`でもwidth更新。

## 3. ファイル切替の間欠的失敗バグ修正

### 課題
左パネルで別の.outファイルを選択しても、アウトライナが切り替わらないことが時々ある。

### 原因分析
2つの潜在的問題を特定:
1. **クリックガードのタイミング問題**: `f.filePath !== currentFile` ガードで、`currentFile`が`notesFileListChanged`受信時にしか更新されないため、タイミングにより送信がスキップされる可能性
2. **openFile失敗時のサイレント無応答**: `fileManager.openFile()`が`null`返却時（ファイル読み込みエラー）、webviewに何も返さずUIが不整合状態になる

### 対策（実装）
- **`notes-file-panel.js`**: クリックハンドラで`currentFile`を即時更新（`currentFile = f.filePath`）。二重送信防止しつつタイミング依存を解消
- **`notes-message-handler.ts`**: `notesOpenFile`でcontent===null時に`sendFileListWithStructure()`で元のファイルリストを再送信しUI状態を復元

## 4. Daily Notes 空ノード自動追加

### 課題
Daily NotesでScope in後、対象日ノードに子ノードが0個の場合、ユーザーが追加する兄弟ノードがScope外になり消えてしまう。

### 対策（設計）
`ensureDailyNode()`後に日付ノードの子が0個なら空子ノード（`text: ''`）を自動追加。3つのDaily Notesハンドラ全てに適用。

### 対策（実装）
- **`notes-file-manager.ts`**: `addNodeToData`を`private`→public化
- **`notes-message-handler.ts`**: `notesOpenDailyNotes`, `notesNavigateDailyNotes`, `notesNavigateToDate`の3ハンドラに共通ロジック追加: `dayNode.children.length === 0`判定 → `addNodeToData(data, dayNodeId, '', 'last')`

## 5. Scope Inホバーアイコン追加

### 課題
Scope in方法がAlt+バレットクリックまたは右クリックメニューのみで発見性が低い。

### 対策（設計）
- 各ノード行のバレット左にscope-btn要素（20px幅）を常時確保
- ホバー時のみアイコン表示（opacity transition）
- クリックで`setScope()`呼び出し

### 対策（実装）
- **`outliner.css`**: `.outliner-scope-btn`スタイル追加（opacity: 0→hover時0.35→btn hover時0.8）
- **`outliner.js`**: `createNodeElement()`にscope-btn要素をindentとbulletの間に追加。右矢印SVGアイコン、click → `setScope({ type: 'subtree', rootId: node.id })`
