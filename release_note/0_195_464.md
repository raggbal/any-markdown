# v0.195.464 — Outliner インライン装飾バグ修正

## 課題

### TASK1: Cmd+B がVSCodeサイドバーを切り替える / `**文字*` が斜体になる
- Cmd+Bで太字適用時、VSCodeのExplorerサイドバーも連動して開閉してしまう
- `**文字*` と入力した瞬間、italicの正規表現 `\*([^*]+)\*` が `*文字*` (位置1〜) にマッチし、bold完成前に斜体表示になる

### TASK2: Cmd+I でカーソルが末尾に飛ぶ / Enterで不正分割
- `abcd` の `bc` を選択して Cmd+I → `bc` が斜体になるがカーソルが `d` の後ろに移動
- その状態でEnter → `a*bc` と `*d` に分割される（マーカーが壊れる）

### TASK3: パターン入力 `*bc*` でも同じカーソル飛び・Enter不正分割
- `a*bcd` の `c` の後に `*` を入力 → `a*bc*d` でbcが斜体になるがカーソルが末尾に飛ぶ
- Enter分割も同じく `a*bc` / `*d` になる

### TASK4: 上記はbold/italic/strikethrough/inline code 全てで発生

## 原因（根本原因）

**レンダリング後テキストとソーステキストのオフセット不一致**

`renderInlineText()` がマーカー (`*`, `**`, `~~`, `` ` ``) をHTMLタグに変換すると、`textContent` からマーカー文字が消失する。

例: ソース `a*bc*d` (6文字) → レンダリング `a<em>bc</em>d` → textContent `abcd` (4文字)

- `getCursorOffset()` はレンダリング後テキスト（4文字）でのオフセットを返す
- `node.text` はソーステキスト（6文字）のまま
- `text.slice(0, offset)` でソーステキストをレンダリング後オフセットで切断 → 不正分割
- `setCursorAtOffset(textEl, 5)` で4文字のテキストに5を指定 → 末尾に飛ぶ

さらに、一度レンダリングすると `getPlainText()` が `textContent` を読むため、以後の入力でマーカーが消失する致命的問題もあった。

## 対策

### 設計: 編集モード/表示モードの分離

| 状態 | レンダリング関数 | マーカー表示 | インライン装飾 |
|------|-----------------|-------------|--------------|
| 編集中 (focus) | `renderEditingText()` | 生テキストで表示 | タグのみハイライト |
| 非編集 (blur) | `renderInlineText()` | HTML変換で非表示 | bold/italic/code/del 全適用 |

**利点:**
- 編集中は `textContent` = ソーステキスト → オフセット一致 → 全バグ解消
- blur時にフル装飾 → 他ノードでは装飾表示
- `getPlainText()` が常にソーステキストを返す → マーカー消失なし

### 実装

1. **`renderEditingText(text)`** 新規追加 — HTMLエスケープ + タグハイライトのみ。マーカーは変換しない
2. **`focus` ハンドラ** — `renderInlineText` → `renderEditingText` に切替。`renderedOffsetToSource()` でカーソル位置をソース空間に変換
3. **`blur` ハンドラ** — `renderInlineText()` でフル装飾に復帰
4. **`input` / `compositionend` ハンドラ** — `renderEditingText()` を使用
5. **`applyInlineFormat()`** — `renderEditingText()` を使用。apply時のカーソル位置を `endOff + 2 * marker.length` に修正
6. **italic正規表現** — `(?<!\*)\*(?!\*)([^*]+)\*(?!\*)` に変更。`**` の一部としての `*` にマッチしない
7. **`e.stopPropagation()`** — Cmd+B/I/E/Shift+S に追加。VSCodeへのイベント伝播を防止
8. **オフセットマッピング関数群** — `stripInlineMarkers()`, `renderedOffsetToSource()`, `sourceOffsetToRendered()`, `buildRenderedToSourceMap()` を追加。focus遷移時の座標変換用
