# v0.195.439 - サイドパネルTOCをサイドバー方式に変更

## 課題

サイドパネルのTOC（目次）表示がトグルボタン方式で、ヘッダー内にTOCトグル・Open Tab・Expand・Closeの各ボタンが詰め込まれていた。メインエディタのサイドバーとUIパターンが異なり、一貫性がなかった。また、TOCの見出しレベルが1-2のみの対応で、h3-h6のインデントが区別されていなかった。

## 原因

サイドパネルのTOCが`side-panel-toc-column`として独自のレイアウト構造を持ち、メインエディタのサイドバー（`aside.sidebar`）とは異なるUIパターンで実装されていた。

## 対策

### 設計

- サイドパネルのTOCカラムをメインエディタのサイドバーと同じUIパターン（`aside`要素 + ハンバーガーメニューで開閉）に統一
- ヘッダーにOutlineボタン（サイドバーアイコン）を追加し、サイドバーが閉じている時のみ表示
- サイドバー内にヘッダー（"Outline" + 閉じるボタン）を配置
- サイドパネル全体を`flex-direction: row`に変更し、サイドバー + エディタコンテナの横並びレイアウトに

### 実装

1. **HTML構造変更** (`editor-body-html.js`):
   - `side-panel-toc-column` → `aside.side-panel-sidebar` に変更
   - サイドバー内に`sidebar-header`（"Outline"タイトル + ハンバーガー閉じボタン）を追加
   - `side-panel-editor-container`で既存のヘッダー+エディタ部分をラップ
   - ヘッダーにOutlineボタン（`side-panel-outline-btn`）を追加

2. **JS変更** (`editor.js`):
   - `sidePanelTocColumn` → `sidePanelSidebar` に変数名変更
   - `sidePanelTocToggle` → `sidePanelOpenOutlineBtn` + `sidePanelSidebarCloseBtn` に分離
   - `openSidePanelSidebar()` / `closeSidePanelSidebar()` ヘルパー関数を追加
   - Outlineボタンクリックでサイドバーを開き、ボタン自体を非表示に
   - ハンバーガーボタンクリックでサイドバーを閉じ、Outlineボタンを再表示

3. **CSS変更** (`styles.css`):
   - `.side-panel`を`flex-direction: row`に変更
   - `.side-panel-sidebar`スタイルを追加（メインサイドバーと同じパターン）
   - `.side-panel-editor-container`でフレックス子要素のレイアウトを管理
   - TOCアイテムのインデントをh1-h6全レベルに対応（`padding-left`で段階的にインデント）
   - TOCフッターのスタイルを調整
