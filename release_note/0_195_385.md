# v0.195.385: Thingsテーマのサイドバーボーダー色調整

## 課題

Thingsテーマにおいて、アウトライン(サイドバー)とエディタの境界線(`--sidebar-border`)が背景色に対してやや目立ちすぎていた。

## 原因

`--sidebar-border: #E5E5EA` がサイドバー背景色 `--sidebar-bg: #F4F5F7` に対してコントラストが強かった。

## 対策

### 設計

サイドバー背景色 `#F4F5F7` に近い色調で、境界を認識できるが主張しすぎない色を選定。

### 実装

- `src/webview/styles.css`: Thingsテーマの `--sidebar-border` を `#E5E5EA` → `#ECEDF0` に変更
