# v0.195.458 - 単一ノードのCmd+C / Cmd+V修正

## 課題

1ノードの文章をCmd+Cした後、Cmd+Vしても何も貼り付かない。複数ノード選択でのコピペは動作する。

## 原因

2つの問題：

1. **Cmd+C**: 単一ノードでテキスト未選択時（カーソルのみ）、ブラウザデフォルトのコピーは選択範囲がないため何もコピーしない。カスタムハンドラは `selectedNodeIds.size > 0` のみ対応。
2. **Cmd+V**: `paste` イベント内で `e.preventDefault()` 後に `document.execCommand('insertText')` を使用していたが、一部環境で動作しない。

## 対策

1. **Cmd+C**: テキスト未選択（`sel.isCollapsed`）時、ノード全体のテキストを `navigator.clipboard.writeText()` でコピー。Cmd+Xも同様に対応（コピー後にノードテキストを空に）。
2. **Cmd+V単一行**: `execCommand` をやめ、カーソル位置でテキストを直接結合してモデル更新 → `renderInlineText` → `setCursorAtOffset` で反映。
