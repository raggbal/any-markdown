# v0.195.463 - Outliner検索モード切替 & インライン装飾ショートカット

## 機能1: 検索モード切替トグルボタン

### 課題

Outlinerの検索ボックスは、マッチしたノードからルートノードまでの祖先を全て表示する「ツリーモード」のみだった。マッチしたノードを頂点として、そのノードと子孫だけを表示する「フォーカスモード」が欲しい。

### 対策

#### 設計

- 検索バーの左側にトグルボタンを追加（🌲 = ツリーモード / 🔍 = フォーカスモード）
- `SearchEngine.search()` に `options.focusMode` パラメータを追加
- フォーカスモード時は祖先ノードを追加しない（マッチノード + 子孫のみ）
- `renderFocusNodes()` でマッチノードをdepth=0として描画、子孫は通常の階層で表示

#### 実装

**outlinerWebviewContent.ts**: 検索バーに `<button class="outliner-search-mode-toggle">` を追加

**outliner-search.js**: `search()` メソッドに第3引数 `options` を追加。`options.focusMode === true` のとき祖先追加をスキップ

**outliner.js**:
- `searchFocusMode` 状態変数を追加
- `setupSearchBar()` にトグルボタンのクリックハンドラを追加
- `executeSearch()` で `{ focusMode: searchFocusMode }` を渡す
- `renderTree()` でフォーカスモード時は `renderFocusNodes()` を呼ぶ
- `renderFocusNodes()`: 直接マッチしたノードを再判定し、各マッチノードをdepth=0として描画
- `renderFocusChildren()`: フォーカスモード用の子孫描画（検索フィルタなし、折りたたみ有効）

**outliner.css**: `.outliner-search-mode-toggle` のスタイル（通常/hover/アクティブ状態）

## 機能2: インライン装飾キーボードショートカット

### 課題

Outlinerのテキストに対してキーボードショートカットでインライン装飾（太字、斜体、取り消し線、インラインコード）を適用できなかった。

### 対策

#### 設計

- `applyInlineFormat(nodeId, textEl, marker)` 関数を新設
- テキスト選択中: 選択範囲をマーカーで囲む。既にマーカーで囲まれている場合は除去（トグル）
- テキスト未選択: カーソル位置にマーカーペアを挿入し、その間にカーソル配置

#### 実装

| ショートカット | 動作 | マーカー |
|---|---|---|
| Cmd+B | 太字 | `**` |
| Cmd+I | 斜体 | `*` |
| Cmd+E | インラインコード | `` ` `` |
| Cmd+Shift+S | 取り消し線 | `~~` |

パターン入力（`**text**`, `*text*`, `~~text~~`, `` `text` ``）は `renderInlineText()` によるリアルタイム描画で既に対応済み。入力した瞬間に装飾が反映される。
