# v0.195.567 リリースノート

## Outliner ノード画像機能

### 課題
Outliner editor のノードに画像を貼り付ける手段がなかった。ノードにメモ画像やスクリーンショットを添付してサムネイル表示したいという要望。

### 対策

#### 新機能: ノード画像のペースト・表示・操作

**データモデル (`outliner-model.js`)**
- ノードモデルに `images: string[]` フィールド追加（相対パス配列）
- `addImage()`, `removeImage()`, `moveImage()` メソッド追加
- `_ensureChildren()` で既存データの後方互換（`images` 未定義 → `[]`）

**画像ペースト (`outliner.js`)**
- `handleNodePaste()` の先頭で `clipboardData.items` から画像ファイルを判定
- 画像があれば `FileReader` → base64 → `host.saveOutlinerImage()` でホストに送信
- 画像がなければ既存のテキストペースト処理にフォールスルー（デグレなし）

**画像表示 (`outliner.js`, `outliner.css`)**
- `createNodeElement()` で subtext の下に `.outliner-images` コンテナ追加
- `renderNodeImages()` でサムネイル描画（60x60px, `object-fit: cover`, `flex-wrap`）
- DOM順序: テキスト → subtext → 画像サムネイル行
- 1行4-5枚、折り返し表示

**画像操作 (`outliner.js`)**
- クリックで選択（青枠表示）、Delete/Backspace で削除
- ダブルクリックで拡大フロート表示（オーバーレイ + Escape/クリック閉じ）
- D&D で同一ノード内の並べ替え（`imageDragState` でノードD&Dと完全分離）

**画像保存先設定**
- `fractal.outlinerImageDefaultDir` (default: `./images`), `fractal.outlinerForceRelativeImagePath` (default: false) を `package.json` に追加
- .out JSON の `imageDir` フィールドで個別設定可（`pageDir` と同パターン）
- 優先順位: .out JSON `imageDir` → VSCode設定 → デフォルト `./images`
- `syncToHostImmediate()` で `imageDir` を付加して永続化

**ホスト側 (`outlinerProvider.ts`)**
- `getOutlinerImageDirPath()`: 画像ディレクトリ解決（JSON優先 → 設定 → デフォルト）
- `saveOutlinerImage`: 画像保存 + 相対パス計算 + displayUri 返却
- `setOutlinerImageDir`: InputBox ダイアログで設定変更
- `localResourceRoots` に画像ディレクトリ追加

**Notes mode 対応 (`notesEditorProvider.ts`, `notes-message-handler.ts`, `notes-host-bridge.js`)**
- `saveOutlinerImage` platformAction: `{pageDir}/images/` に保存（MDページ画像と同一フォルダ）
- Notes mode では画像フォルダ設定メニューを非表示（自動管理）

**ホストブリッジ (`outliner-host-bridge.js`, `notes-host-bridge.js`)**
- `saveOutlinerImage()`, `setOutlinerImageDir()`, `getOutlinerImageDir()` メソッド追加

**設定メニュー (`outliner.js`)**
- `toggleMenuDropdown()` に「Set image directory...」メニュー項目追加（Notes mode 以外）

**WebviewContent (`outlinerWebviewContent.ts`, `notesWebviewContent.ts`)**
- `__outlinerImageBaseUri` を HTML に埋め込み（画像 src 解決用）
- `OutlinerConfig` / `NotesConfig` に `documentBaseUri` プロパティ追加

### テスト結果
- 全915テスト通過、4スキップ、0失敗 — デグレなし
