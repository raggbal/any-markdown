# v0.195.407 - Action Panel（ページ追加機能）

## 課題

Markdownエディタ内から新規ページ（.mdファイル）を作成してリンク挿入する手段がなかった。Notionのようにエディタ内で素早くページを追加し、リンクで結合するワークフローが必要。

## 対策

### 設計

Cmd+N（またはコマンドパレットの「Add Page」）でフローティングAction Panelを表示。3ステップのフロー:

1. **メニュー**: 「ページ追加」/「パス指定でページ追加」から選択（↑↓+Enter）
2. **パス入力**（パス指定時のみ）: オートコンプリート付きテキスト入力
3. **リンク名確定**（共通）: デフォルト=パス（全選択状態）、Enter確定

確定後、カーソル位置に`[linkName](path)`のMarkdownリンクをHTML `<a>`として挿入。

#### メッセージフロー

```
editor.js → host.searchFiles(query)     → editorProvider: workspace.findFiles → fileSearchResults
editor.js → host.createPageAuto()       → editorProvider: pages/{timestamp}.md作成 → pageCreatedAtPath
editor.js → host.createPageAtPath(path) → editorProvider: 指定パスにmd作成 → pageCreatedAtPath
```

#### ファイル名形式（ページ追加）

画像ペーストと同じ`generateUniqueFileName`方式: `{Date.now()}.md`
