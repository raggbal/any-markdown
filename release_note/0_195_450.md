# v0.195.450 - Outlinerタグの日本語対応

## 課題

Outlinerモードで `#タスク` や `@会議` のように日本語（マルチバイト文字）でタグを入力してもタグとして認識されない。英数字（`#tag`、`@meeting`）のみタグになる。

## 原因

タグ正規表現に `\w`（= `[A-Za-z0-9_]`）を使用しており、日本語・韓国語・中国語などのUnicode文字にマッチしない。

## 対策

### 設計

`\w` をUnicode文字プロパティ `[\w\p{L}]` に置換し、`u` フラグを付与してUnicode対応にする。

### 実装（2箇所）

1. **outliner.js `renderInlineText`** — HTML表示用のタグspan生成正規表現
2. **outliner-model.js `parseTags`** — データモデル用のタグ抽出正規表現

変更前: `/(?<!\w)([#@]\w[\w-]*)/g`
変更後: `/(?<![&#\w\p{L}])([#@][\w\p{L}][\w\p{L}-]*)/gu`
