# v0.195.457 - Shift+Enterで子ノード追加

## 課題

Shift+Enterで現在ノードの子として新しいノードを追加したい。既に子がある場合は子リストの先頭に追加。（Enterは従来通り常にすぐ下に兄弟ノードを追加）

## 対策

### 設計

`handleShiftEnter()` 関数を新設。Enterキーハンドラで `e.shiftKey` を判定し分岐。

### 実装

| 状態 | 動作 |
|---|---|
| カーソル位置にテキストあり | カーソルより後のテキストが子ノードに分離 |
| 子ノードが既にある | 子リストの先頭に追加（`addNodeAtStart`） |
| 折りたたみ中 | 自動展開して子ノード追加 |
| タスクノード | 子もタスクノード（未チェック）として継承 |
