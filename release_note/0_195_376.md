# v0.195.376: ツールバー・コマンドパレットにMermaid/Mathブロック直接挿入ボタンを追加

## 課題

Mermaidブロックと Mathブロックの作成方法が限られていた:
- `` ```mermaid `` / `` ```math `` パターン入力 + Enter
- コードブロック作成後、言語セレクターから変換

ツールバーやコマンドパレット（Cmd+/）から直接作成する手段がなかった。

## 対策

### 設計

ツールバーとコマンドパレットは既に `dispatchToolbarAction()` 共通関数を使用しているため、ここに `mermaid` / `math` ケースを追加することで両方から利用可能にする。

ブロック作成には既存の `convertToSpecialBlock()` を再利用し、コードの重複を避ける。`convertToSpecialBlock()` は v0.195.375 で click/input/focusout の3つのイベントハンドラが登録済みのため、追加のハンドラ登録は不要。

### 実装

#### `src/webview/editor.js`

1. **LUCIDE_ICONS**: mermaid（フローチャート風）と math（Σ記号）のSVGアイコンを追加

2. **dispatchToolbarAction()**: `case 'mermaid'` / `case 'math'` を追加
   - 空の `<pre data-lang="mermaid|math">` + `<code><br></code>` を作成
   - 現在行が空なら置換、テキストありなら直後に挿入
   - `convertToSpecialBlock(pre, action)` で特殊ラッパーに変換
   - `nextSibling` を事前記録し、変換後に wrapper を取得
   - `enterSpecialWrapperEditMode(wrapper, 'start')` で編集モードに入る

3. **COMMAND_PALETTE_ITEMS**: blocks グループに mermaid / math の2項目を追加（19→21項目）

#### `src/webviewContent.ts`

ツールバーの block グループに `data-action="mermaid"` と `data-action="math"` ボタンを追加（codeblock と hr の間）。

#### `src/i18n/messages.ts` + 7ロケール

`WebviewMessages` 型に `mermaidBlock` / `mathBlock` を追加。全7言語の翻訳を追加。

#### `test/specs/command-palette.spec.ts`

アイテム数の期待値を 19 → 21 に更新（2箇所）。

### 特殊ラッパー作成パス（4箇所に拡大）

| パス | 関数/箇所 | トリガー |
|---|---|---|
| 初期レンダリング | `setupMermaidDiagrams()` / `setupMathBlocks()` | `renderFromMarkdown` |
| パターン変換 | `checkBlockPatterns()` 内 | `` ```mermaid `` / `` ```math `` + Enter |
| 言語セレクター | `convertToSpecialBlock()` | コードブロックの言語タグで選択 |
| **ツールバー/パレット** | **`dispatchToolbarAction()` → `convertToSpecialBlock()`** | **ツールバーボタン or コマンドパレット** |
