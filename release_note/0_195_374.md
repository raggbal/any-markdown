# v0.195.374 リリースノート

## 変更概要

ツールバーデフォルトモードの変更、ショートカットキーの修正、README全面改訂。

---

## 1. toolbarMode のデフォルトを `full` → `simple` に変更

### 課題
Action Palette (`Cmd+/`) の導入により、全ボタン表示のフルツールバーが冗長になった。新規ユーザーにはシンプルな見た目がデフォルトとして適切。

### 対策
- `package.json` の `any-markdown.toolbarMode` の `default` を `"full"` → `"simple"` に変更
- フルツールバーが必要なユーザーは設定で `"full"` に戻せる

---

## 2. Open in Text Editor のショートカットを `Cmd+,` → `Cmd+Shift+.` に変更

### 課題
`Cmd+,` は VS Code の標準ショートカット（Settings を開く）と競合。`when` 条件で Any Markdown エディタ時のみ発動するが、ユーザーが Settings を開きたい時に混乱する。

### 対策
- `package.json` keybinding: `cmd+,` / `ctrl+,` → `cmd+shift+.` / `ctrl+shift+.`
- `webviewContent.ts` ツールバー tooltip: `Cmd+,` → `Cmd+Shift+.`
- `Cmd+.` (Source Mode) と `Cmd+Shift+.` (Open in Text Editor) のペアで覚えやすい設計

---

## 3. README.md 全面改訂

### 課題
README が最新の機能（Action Palette、toolbarMode、ショートカット変更）を反映していなかった。また、Keyboard Shortcuts セクションのショートカットが誤っていた（Toggle Source Mode と Open in Text Editor が逆）。

### 対策
- **Important Changes セクション追加**: Action Palette の紹介、toolbar simple モードがデフォルトになった旨
- **General Shortcuts に統合**: Commands 配下の Keyboard Shortcuts を Special Operations > General Shortcuts に移動
- **ショートカット修正**:
  - `Cmd+/` → Open Action Palette（新規追加）
  - `Cmd+.` → Toggle Source Mode（修正）
  - `Cmd+Shift+.` → Open in Text Editor（修正）
- **Settings テーブル更新**: toolbarMode デフォルト表記を `simple` に
- **h2 に絵文字追加**: 各セクション見出しに絵文字を付与
- **スクリーンショット更新**: 新デザインのスクリーンショットとAction Paletteのスクリーンショットを追加
