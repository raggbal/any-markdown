# v0.195.451 - Outliner複数ノード選択・コピー＆ペースト

## 課題

Outlinerモードで以下ができない：
- マウスドラッグ、Shift+Click、Shift+↑↓での範囲選択
- 範囲選択してのCmd+C / Cmd+V
- mdモードのリストでは普通にできていた

## 原因

Outlinerは各ノードが個別の `contenteditable` div であるため、ブラウザ標準の複数要素横断選択が機能しない。mdモードは単一contenteditableなのでブラウザ標準で動作する。

## 対策

### 設計

カスタムの複数ノード選択システムを実装。選択状態（`selectedNodeIds` Set）とアンカー（`selectionAnchorId`）を管理し、各操作に対応するハンドラを追加。

### 実装

| 操作 | 実装 |
|---|---|
| Shift+↑/↓ | `selectRange(anchorId, targetId)` で範囲選択、`.is-selected` CSSクラスで青ハイライト |
| Shift+Click | `mousedown` イベントで `selectRange` を実行 |
| Cmd+A | 全ノードを選択 |
| Cmd+C | `getSelectedText()` でタブインデント付きテキストをクリップボードに書き込み |
| Cmd+X | コピー後に `deleteSelectedNodes()` |
| Cmd+V | `pasteNodesFromText()` でインデント構造を解析してノード階層を復元 |
| Backspace/Delete | 選択ノード一括削除 |
| 通常キー/クリック | 選択クリア |
