# v0.195.368: Simple Toolbar Mode 追加

## 課題

ツールバーにボタンが多数並んでおり、Cmd+/（コマンドパレット）で操作する場合にはヘッダー領域が冗長になる。ミニマルな表示モードが欲しい。

## 対策

### 設計

- VSCode Settings で `any-markdown.toolbarMode` を `"full"` / `"simple"` で切替可能にする
- `data-toolbar-mode` 属性を `<html>` に付与し、CSS のみで表示を切り替える（JS変更不要）
- 既存の `data-theme` と同じ設定注入パターンを踏襲

### 実装

**Simple モードの表示:**
- 左側: open outline, undo, redo
- 右側: open text editor, toggle source mode（`margin-left: auto` で右端固定）
- 背景: `transparent`
- ヘッダー区切り線 (`border-bottom`): なし
- グループ間セパレーター (`::before`): なし
- スクロールボタン: 非表示

**非表示グループ:**
- `data-group="inline"` (bold, italic, strikethrough, code)
- `data-group="block"` (headings, lists, quote, codeblock, hr)
- `data-group="insert"` (link, image, imageDir, table)

**変更ファイル:**
1. `package.json` — `any-markdown.toolbarMode` 設定定義
2. `src/webviewContent.ts` — `EditorConfig` に `toolbarMode` 追加、`<html>` に `data-toolbar-mode` 属性注入
3. `src/editorProvider.ts` — Settings から `toolbarMode` 読み取り
4. `src/webview/styles.css` — `[data-toolbar-mode="simple"]` CSS ルール追加
5. `test/build-standalone.js` — テスト用HTMLに `data-toolbar-mode="full"` 追加
