# 要件定義書 — Outliner ページパスコピー機能

## 概要

Outliner editorのページノード（`isPage === true`）に対して、そのページの .md ファイルのフルパス（絶対パス）をクリップボードにコピーする機能を追加する。

---

## 機能要件

### コンテキストメニュー（右クリック）

| No | 要件 |
|---|---|
| CPP-1 | ページノードの右クリックメニューに「Copy Page Path」を追加する |
| CPP-2 | 非ページノードでは「Copy Page Path」を表示しない |
| CPP-3 | 実行すると、そのページの .md ファイルのフルパス（絶対パス `{pagesDir}/{pageId}.md`）をシステムクリップボードにコピーする |
| CPP-4 | ショートカットキー表示: `Cmd+Shift+C`（Mac） / `Ctrl+Shift+C`（Windows） |

### キーボードショートカット

| No | 要件 |
|---|---|
| CPP-10 | `Cmd+Shift+C`（Mac）/ `Ctrl+Shift+C`（Windows）でフォーカス中のページノードのパスをコピー |
| CPP-11 | フォーカス中のノードがページノードでない場合は何もしない |

### 複数選択時の動作

| No | 要件 |
|---|---|
| CPP-20 | 複数ノード選択中に `Cmd+Shift+C` を実行すると、選択ノードの中のページノードの .md ファイルパスを改行区切りでクリップボードにコピーする |
| CPP-21 | 選択ノードの中にページノードでないノードが混在している場合、ページノードのパスのみを抽出する（非ページノードは無視） |
| CPP-22 | 選択ノードの中にページノードが1つもない場合は何もしない |
| CPP-23 | パスの順序はDOM表示順（上から下）に従う |

### 複数選択時のコンテキストメニュー

| No | 要件 |
|---|---|
| CPP-30 | 複数ノード選択中に右クリックした場合、選択ノードの中にページノードが1つ以上あれば「Copy Page Path」を表示する |
| CPP-31 | 複数選択中の「Copy Page Path」実行時は、CPP-20〜CPP-23と同じ動作をする |
| CPP-32 | 選択ノードの中にページノードが1つもない場合は「Copy Page Path」を表示しない |

### クリップボード出力形式

| No | 要件 |
|---|---|
| CPP-40 | 単一ノード: フルパス1行のみ（例: `/path/to/pages/uuid.md`） |
| CPP-41 | 複数ノード: フルパスを改行（`\n`）区切りで結合（例: `/path/to/pages/uuid1.md\n/path/to/pages/uuid2.md`） |

---

## 対象環境

| 環境 | 対応 |
|---|---|
| VSCode拡張（.out単体） | 対応 |
| Notes editor（.note経由） | 対応 |
| Electron版 | 対応（HostBridgeに追加） |
| スタンドアロンテスト | 対応（テスト用HostBridgeに追加） |

---

## 非機能要件

- 既存のクリップボード操作（Cmd+C / Cmd+X）に影響を与えない
- 既存のコンテキストメニュー項目に影響を与えない
- i18n対応（メニューラベルのローカライズキー追加）
